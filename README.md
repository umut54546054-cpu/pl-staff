pl-staff
Yüksek performans odaklı, asenkron tabanlı yetkili takip ve raporlama sistemi.

🛡️ Proje Özeti
pl-staff, Minecraft sunucularında yetkili kadrosunun aktivitelerini ölçümlemek için geliştirilmiş, sunucu performansını (TPS) ön planda tutan profesyonel bir yönetim eklentisidir.

🚀 Temel Özellikler
Asenkron Mimari: İşlemler main-thread'i yormadan, arka planda asenkron olarak gerçekleşir.

Discord Entegrasyonu: Tüm raporlar anlık olarak Webhook aracılığıyla Discord'a iletilir.

GUI Yönetimi: Yetkili performansını izleyebileceğiniz dinamik arayüz.

Sıfır Lag: Optimize edilmiş veri yazma işlemleri ile yüksek oyuncu kapasiteli sunucularda dahi stabilite.

⚙️ Kurulum
.jar dosyasını plugins klasörünüze yerleştirin.

Sunucuyu başlatarak klasörün oluşmasını sağlayın.

config.yml üzerinden Discord Webhook adresinizi tanımlayın.

/pl-staff reload komutuyla yapılandırmayı devreye alın.
